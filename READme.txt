Instructions to run my converter:
	1. Place your input text files in folder 
	2. Run this line: python3 converter.py
	3. Type one of text files with (.txt) Eg: testing.txt
	4. Insert type of input  Eg: UTF-8, UTF-32, etc.
	5. Insert type of expecting output  Eg: UTF-8, Baudot code, Morse code, etc.
	6. The program returns output.txt you expected (output.txt will overwrite if you change input and output)
